class DINOv2Teacher:
    def __init__(self, model_name="...", device="cuda"):
        self.model = AutoModel.from_pretrained(model_name, ...)
        self.model.to(device).eval()
        self.device = device
        self.transform = transforms.Compose([...])

    @torch.no_grad()
    def encode_batch(self, images):
        tensors = [self.transform(img) for img in images]
        batch = torch.stack(tensors).to(self.device)
        with torch.amp.autocast(self.device):
            out = self.model(batch)
        emb = out.last_hidden_state
        return [e.cpu().numpy() for e in emb]
```

class DINOv3FTTeacher:
    """Fine-tuned DINOv3 ViT-H+ as teacher. Loads base model + LoRA adapter."""
```

**__init__(self, adapter_path="dino_finetune/output/best_adapter", device="cuda"):**
- Import at function level: `from peft import PeftModel` and `from transformers import AutoModel, AutoImageProcessor`
- Load base DINOv3 model: `AutoModel.from_pretrained("facebook/dinov3-vith16plus-pretrain-lvd1689m", torch_dtype=torch.bfloat16, attn_implementation="sdpa", low_cpu_mem_usage=True)`
- Load LoRA adapter on top: `PeftModel.from_pretrained(base_model, adapter_path)`
- Move to device, set eval, freeze all params
- Load image processor: `AutoImageProcessor.from_pretrained("facebook/dinov3-vith16plus-pretrain-lvd1689m")`
- Store self.model, self.processor, self.device
- Build self.transform using the processor's values (or just store processor for direct use)
- Log: `"DINOv3FTTeacher: loaded adapter from {adapter_path}, output_dim=1280"`
- **IMPORTANT:** Handle missing adapter gracefully -- if adapter_path does not exist, raise a clear error: "DINOv3FTTeacher: adapter not found at {adapter_path}. Run dino_finetune/train_dino.py first."

**encode_batch(self, images: list[np.ndarray | Image.Image]) -> list[np.ndarray | None]:**
- Same interface as TrendyolEmbedder.encode_batch and DINOv2Teacher.encode_batch
- Convert images to PIL RGB if needed
- Process through AutoImageProcessor (handles resize, normalize)
- Forward pass with torch.amp.autocast, extract CLS token: `outputs.last_hidden_state[:, 0]`
- L2-normalize embeddings: `F.normalize(cls_emb, dim=1)`
- Return list of numpy arrays, each shape (1280,)
- Decorate with @torch.no_grad()

**Key differences from DINOv2Teacher:**
- Uses AutoImageProcessor instead of manual transforms (DINOv3 may need different normalization)
- Loads LoRA adapter via PeftModel (DINOv2Teacher loads a pre-trained model directly)
- Output dimension is 1280 (vs 256 for DINOv2Teacher which uses the Trendyol 256d variant)
- Uses bf16 dtype for VRAM efficiency
- CLS token extraction needs to account for 4 register tokens in DINOv3 (but CLS is still index 0)

**Do NOT modify any other part of prepare.py.** Only add the new class after DINOv2Teacher.
