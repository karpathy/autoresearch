# report-design
Optimize the HTML research report for professional quality and narrative coherence.

## Files
- `workflow.yaml` | targets, metric, run command | human-configured
- `program.md` | research strategy and constraints | human-written
- `evaluate.py` | report quality evaluator | fixed
- `../../.github/skills/report-generator/templates/` | templates being optimized | agent-edited

## Run
1. Read `program.md` for strategy
2. Read `workflow.yaml` for targets and metric
3. Follow `autonomous-iteration` skill in `../../.github/skills/autonomous-iteration/SKILL.md`
