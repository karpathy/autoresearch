# Benchmarks

`archive/` stores versioned benchmark snapshots that are useful project history.

`generated/` is for local ad-hoc benchmark and probe output. It is ignored by Git so we can keep the repo root clean without losing local experiment logs.

Useful summaries:

- `SUMMARY_2026-03-15.md` is the detailed benchmark table and experiment recap for the current tuning round.
- `KEY_FINDINGS.md` is the short list of durable conclusions worth carrying forward when revisiting performance work.
